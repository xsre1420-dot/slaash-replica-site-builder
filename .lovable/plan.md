
# خطة تحسين صفحة تفاصيل المنتج

## ملخص التحسينات
تطبيق 15 تحسين على صفحة تفاصيل المنتج لتتوافق مع نفس مستوى جودة صفحة المتجر المُحسّنة.

---

## التحسينات المقترحة

### 1. Skeleton Loading بدل شاشة التحميل البسيطة
- استبدال دائرة التحميل الحالية بـ Skeleton يحاكي شكل الصفحة الفعلي (صورة + عنوان + سعر + أزرار)
- تجربة أسرع بصرياً للمستخدم

### 2. تحسين Header مع زر المفضلة والمشاركة
- اضافة زر قلب (مفضلة) وزر مشاركة في الـ Header
- ربط المفضلة بـ `useFavorites` hook الموجود
- استخدام Web Share API للمشاركة

### 3. معرض صور محسّن مع Thumbnails
- اضافة شريط صور مصغرة أسفل الصورة الرئيسية
- مؤشر نقاط للصور (dots indicator)
- تأثير zoom عند النقر على الصورة

### 4. شارات ذكية على الصور
- شارة "جديد" للمنتجات الحديثة (اقل من 7 ايام)
- شارة "آخر X قطع" للمخزون المنخفض
- شارة الخصم مع النسبة المئوية
- شارة "نفذ المخزون"

### 5. عرض السعر الأصلي والخصم
- عرض السعر الأصلي مشطوب اذا كان هناك خصم
- شارة نسبة الخصم بجانب السعر
- تنسيق أفضل للأسعار

### 6. تحسين اختيار المقاس واللون
- تصميم أنيق مع حركات انتقالية سلسة
- عرض اسم اللون المحدد
- تأثير بصري واضح عند التحديد مع علامة صح

### 7. عداد الكمية قبل الإضافة للسلة
- اضافة عداد كمية (+/-) فوق زر الإضافة للسلة
- الحد الأقصى مرتبط بالمخزون المتاح

### 8. Toast تأكيد عند الإضافة للسلة
- رسالة تأكيد جميلة عند الإضافة باستخدام sonner
- تتضمن اسم المنتج والخيارات المحددة

### 9. شريط ضمانات (شحن - ضمان - إرجاع)
- ثلاث أيقونات: شحن سريع، ضمان الجودة، سياسة الإرجاع
- تصميم أنيق بخلفية رمادية فاتحة

### 10. Social Proof (عدد المشاهدين)
- عرض "X شخص يشاهد هذا المنتج الآن"
- عرض عدد مرات الشراء إن وجد

### 11. تحسين قسم الوصف
- تصميم أجمل مع خلفية وحدود
- دعم عرض النص بشكل منسق

### 12. تحسين قسم التقييمات
- تصميم أنظف مع فصل بصري أفضل
- انيميشن عند فتح نموذج التقييم

### 13. تحسين المنتجات المقترحة
- تصميم بطاقات متوافق مع ProductCard في المتجر
- زر اضافة سريعة للسلة

### 14. تحسين شريط السلة السفلي
- تأثير ظهور سلس (slide up animation)
- تصميم أكثر وضوحاً

### 15. Lazy Loading للصور
- تحميل كسول لصور المنتجات المقترحة
- تأثير blur-up عند تحميل الصور

## التفاصيل التقنية

### الملفات التي سيتم تعديلها:

| الملف | التعديل |
|-------|---------|
| `src/pages/ProductDetails.tsx` | اعادة بناء الصفحة بالكامل مع Skeleton، عداد الكمية، الشارات، الضمانات، toast، social proof |
| `src/components/product-details/ProductHeader.tsx` | اضافة أزرار المفضلة والمشاركة |
| `src/components/product-details/ProductImages.tsx` | اضافة thumbnails ومؤشر النقاط وتأثير zoom |
| `src/components/product-details/AddToCartButton.tsx` | تحسين التصميم مع حالة disabled للمنتج النافذ |
| `src/components/product-details/SuggestedProducts.tsx` | تحسين بطاقات المنتجات مع lazy loading وزر اضافة سريعة |
| `src/components/product-details/ExpandableSection.tsx` | انيميشن سلسة للفتح والاغلاق |

### المكتبات المستخدمة (موجودة بالفعل):
- `sonner` للـ toast notifications
- `lucide-react` للأيقونات
- `embla-carousel-react` عبر shadcn Carousel
- `useFavorites` hook الموجود
